# Typography
## Standard typography and text

All fonts are a variation of Arial.

### Color
Text should only ever be black or white.
<Preview withToolbar>
    <Story name="colors">
        <div>
            <Text>Black text</Text>
            <Text white>

                    White text
                </Highlighter>
            </Text>
        </div>
    </Story>
</Preview>


### Highlighter
For `Text` typography, you can highlight in any non-b/w theme color. Only highlight black text.
