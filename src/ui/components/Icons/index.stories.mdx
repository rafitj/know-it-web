# Icons

## Standard icons

Grommet-icons are the standard icons. Visit https://icons.grommet.io/.
Icons **must** be wrapped in `IconWrap` which is a wrapper of `reflexbox` `Box` but with color.

<Preview withToolbar>
  <Story name="Standard">
    <div>
      <IconWrap m={3}>
        <TextAlignCenter />
      </IconWrap>
      <IconWrap m={3}>
        <TextAlignFull />
      </IconWrap>
      <IconWrap m={3}>
        <TextAlignLeft />
      </IconWrap>
      <IconWrap m={3}>
        <TextAlignRight />
      </IconWrap>
    </div>
  </Story>
</Preview>

### Combining

Icons can also be used with `Cards` and `Buttons`

<Preview withToolbar>
  <Story name="Composites">
    <div>
      <Card>
        <IconWrap>
          <Edit />
        </IconWrap>
      </Card>
      <SecondaryButton icon={<Edit />} text="Edit Profile" />
    </div>
  </Story>
</Preview>

### Color

Icons should usually only be black or white but you can add brand colors to `IconWrap`

<Preview withToolbar>
  <Story name="Colors">
    <div>
      <IconWrap m={3}>
        <FacebookOption />
      </IconWrap>
      <IconWrap m={3} iconColor="blue">
        <FacebookOption />
      </IconWrap>
    </div>
  </Story>
</Preview>
